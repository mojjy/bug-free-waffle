## Fizz Buck Sample
> This project is built using - angular (gulp,and webpack)

## Installation

```bash
$ npm install -g typescript
$ npm start
```

## Dev mode
```bash
$ gulp dev-server 
```

## Production Build
```bash
$ npm run build
```

## Running Unit Tests
```bash
$ npm test
```

## Running End-to-End Tests
```bash
# make sure you have a running app
$ npm run e2e
```
